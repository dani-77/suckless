# suckless
Suckless tools config and  install script.

Simple suckless tools install scripts for Debian/Arch based linux distributions.

After `git clone https://github.com/dani-77/suckless`

Run `chmod +x *_install.sh`

And then run your script.
Example: `sh debian_install.sh`

Included are 2 extra scripts for creation of .profile or .xinitrc.

Enjoy.
